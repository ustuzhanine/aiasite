## 3. Логика слоев видео-фона (z-index система)

```css
z-index: 0   → Фоновое видео (самый нижний слой)
z-index: 1   → Матовое стекло (backdrop-filter: blur)
z-index: 2   → Текст, заголовки и кнопки
z-index: 10  → Шапка сайта (всегда поверх контента)
z-index: 20  → Бургер-меню (когда открыто)
```

### Структура hero-секции:
```html
<section class="hero">
  <div class="video-background">     <!-- z-index: 0 -->
    <video autoplay muted loop>...</video>
  </div>
  <div class="glass-overlay"></div>  <!-- z-index: 1 -->
  <div class="hero-content">         <!-- z-index: 2 -->
    <h1>Заголовок</h1>
    <button>Кнопка</button>
  </div>
</section>
```

## 4. Карусель партнеров - технические требования

### Функциональность:
- ✅ Автоматическое горизонтальное движение
- ✅ Бесконечный цикл (seamless loop)
- ✅ Пауза анимации при наведении курсора
- ✅ Адаптивность: разное количество логотипов на экран
- ✅ Плавная CSS анимация без JavaScript

### CSS Animation подход:
```css
@keyframes scroll {
  0% { transform: translateX(0); }
  100% { transform: translateX(-50%); }
}

.carousel-track {
  animation: scroll 30s linear infinite;
}

.carousel-track:hover {
  animation-play-state: paused;
}
```

## 5. Адаптивная верстка - breakpoints

```css
/* Мобильные устройства */
@media (max-width: 768px) {
  - Бургер-меню вместо горизонтальной навигации
  - Вертикальное расположение элементов
  - Упрощенные анимации и эффекты
  - Увеличенные размеры кнопок для touch
}

/* Планшеты */
@media (max-width: 1024px) {
  - Средние отступы и размеры
  - Адаптация сетки услуг (2 колонки)
  - Оптимизация карусели партнеров
}

/* Десктоп */
@media (min-width: 1025px) {
  - Полная функциональность
  - Максимальные отступы
  - Hover эффекты
}
```

## 7. Технические детали

### Видео оптимизация:
- Формат: MP4 (основной) + WebM (для поддержки)
- Разрешение: 1920x1080 максимум
- Продолжительность: 10-30 секунд для loop
- Размер файла: до 5MB для быстрой загрузки

### Performance:
- Preload критических ресурсов
- Lazy loading для изображений ниже fold
- CSS и JS минификация
- Оптимизация изображений (WebP формат)

### Браузерная совместимость:
- Chrome, Firefox, Safari, Edge (последние версии)
- Fallback для старых браузеров
- Graceful degradation для backdrop-filter
